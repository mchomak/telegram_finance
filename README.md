# Expense Tracker Bot

Telegram-бот для учёта личных и совместных расходов с голосовым вводом, GPT-категоризацией и экспортом в Excel.

## Быстрый старт

**1. Клонируйте репозиторий**

```bash
git clone <repo-url>
cd telegram_finance
```

**2. Создайте файл с переменными окружения**

```bash
cp .env.example .env
```

Откройте `.env` и заполните значения:

| Переменная | Что указать |
|---|---|
| `BOT_TOKEN` | Токен бота из [@BotFather](https://t.me/BotFather) |
| `OPENAI_API_KEY` | Ключ OpenAI API |
| `WHISPER_MODEL` | Размер модели Whisper: `tiny`, `base`, `small`, `medium` (по умолчанию `base`) |
| `POSTGRES_USER` | Имя пользователя БД (придумайте сами, например `expensebot`) |
| `POSTGRES_PASSWORD` | Пароль БД (придумайте сами) |
| `POSTGRES_DB` | Имя базы данных (например `expensebot`) |
| `DATABASE_URL` | Замените `expensebot` и `secret` на ваши `POSTGRES_USER` и `POSTGRES_PASSWORD` |

**3. Запустите**

```bash
docker compose up -d
```

Бот автоматически дождётся готовности базы данных и применит миграции.

Проверить логи:

```bash
docker compose logs -f bot
```

## Как записывать расходы

Просто отправьте боту **голосовое сообщение**. Бот понимает свободную речь:

- «Потратил 450 рублей в Пятёрочке на продукты»
- «Купили пиццу за 1200, Серёга должен половину»
- «Такси домой, не помню сколько, с Машей»
- «Кофе и круассан, где-то 350»

После расшифровки бот покажет, что он понял, и предложит подтвердить или поправить сумму, категорию, участников и дату.

**Кнопки на клавиатуре:**

| Кнопка | Действие |
|---|---|
| 📋 История | Последние траты с пагинацией и сводкой долгов |
| 📤 Экспорт | Скачать `.xlsx` за текущий / прошлый месяц или за всё время |
| ⚙️ Настройки | Управление категориями |

**Команды:**

- `/start` — первый запуск, создание начальных категорий
- `/debts` — кто и сколько должен

## Обновление

```bash
git pull
docker compose up -d --build
```

Миграции применяются автоматически при каждом старте.
